# Publicar cursos y exámenes JSON

1. Crea o edita un examen JSON con el constructor del profesor.
2. Sube el archivo a `data/exams/`.
3. Agrega su ruta relativa en `data/catalog.json`, por ejemplo `./data/exams/fisica-mru.json`.
4. Haz commit y sube los cambios a GitHub.
5. Espera la actualización de GitHub Pages.
6. Recarga la plataforma.

Los alumnos solo ven los exámenes publicados en estos archivos. Los borradores locales del profesor no se publican automáticamente.
